Search Bar: To Search Specific Cuisine Types in a Particular City:
On the nav bar, just enter a city name and a cuisine type and then press the search button
Then the application looks at YELP AP database and returns an unordered list of
restaurants serving the specified cuisine in that particular city. Each list item contains a
restaurant name. ratings. and its address. If the users would like to view details of a
restaurant, they can click on the restaurant name and the application will displav full
detail of the restaurant such as its name, ratings, phone number, photos, operating hours
and etc. Or the user can view the restaurant on Yelp website by clicking on the provided link.
2. Weather Forecast Sidebar:
When searching a cuisine type in a particular city, the weather forecast feature displavs
the current weather condition of that particular city by retrieving data from weather API.
Currency Converter Sidebar:
The currency converter service uses the Forex-Pvthon Converter API. The users can select
a from-currency and a to-currency abbreviation and an amount. The converter takes the from-currency
amount and converts it to the to-currency amount.
4. Favorite List Sidebar:
The application has a favorite-list sidebar. The user can save a restaurant in the favorite.
list for future reference or as a bucket list. The favorite-list stores the required restaurant
information in a database and only displays the restaurant name, city and state to the
user. By clicking on the restaurant name, full details of the restaurant will be displayed.
